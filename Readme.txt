1. This program supports only Python 3.9, 3.10, and 3.11 because newer versions do not support OpenCV.  
2. Before using the program, install the required libraries by opening the Command Prompt and running:  
   ```
   pip install -r requirements.txt
   ```  
3. To run the program, execute the `app.py` file using the command:  
   ```
   python app.py
   ```  
   After that, the system will generate an IP link. Use that link to access the application.


1. โปรแกรมนี้รองรับเฉพาะ Python 3.9, 3.10 และ 3.11 เท่านั้น เนื่องจากเวอร์ชันใหม่กว่านี้ไม่รองรับ OpenCV  
2. ก่อนเริ่มใช้งาน ให้ติดตั้งไลบรารีโดยเปิด Command Prompt และรันคำสั่ง:  
   ```
   pip install -r requirements.txt
   ```  
3. เมื่อต้องการรันโปรแกรม ให้รันไฟล์ `app.py` ด้วยคำสั่ง:  
   ```
   python app.py
   ```  
   จากนั้น ระบบจะสร้างลิงก์ IP ขึ้นมา ให้ใช้ลิงก์ดังกล่าวเพื่อเข้าถึงแอปพลิเคชัน
